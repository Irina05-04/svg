What is it SVG?

SVG is one of the types of graphics, which is formed using geometric objects: lines, circles, rectangles, curves and other elements.

Basic shapes of SVG images

  • simple line
  • broken line
  • rectangle
  • circle
  • ellipse
  • polygon

Animation SVG

CSS Animation SVG with keyframes

@keyframes change_fill {
	0% { fill: #fbff7f; }
	75% { fill: #97fbec; }
	100% { fill: #66dc6e; }
}
svg {
	fill: #1bce4f;
	animation: change_fill 1s;
	animation-iteration-count:infinite;
}
